## How to Generate

These figures should be generated by rendering the Quarto exercise file with R:

```bash
cd /path/to/ims
quarto render exercises/_21-ex-inference-paired-means.qmd
```

Alternatively, run the R code blocks directly and save the outputs using ggsave() to the appropriate paths.

## Dependencies

- R with tidyverse, openintro, infer packages
- Quarto (for rendering QMD files)
