# PreTeXt Book Structure

This directory contains the PreTeXt book structure for Introduction to Modern Statistics (2e).

## Main Project File

- `project.ptx` - Main PreTeXt project file defining the book structure

## Structure

The book is organized into the following parts:

### Preface
- `preface.ptx` - Book preface (from `preface.qmd`)

### Part I: Introduction to data
1. Hello data (`chapters/ch01-hello-data.ptx` from `data-hello.qmd`)
2. Study design (`chapters/ch02-study-design.ptx` from `data-design.qmd`)
3. Applications: Data (`chapters/ch03-data-applications.ptx` from `data-applications.qmd`)

### Part II: Exploratory data analysis
4. Exploring categorical data (`chapters/ch04-exploring-categorical.ptx` from `explore-categorical.qmd`)
5. Exploring numerical data (`chapters/ch05-exploring-numerical.ptx` from `explore-numerical.qmd`)
6. Applications: Explore (`chapters/ch06-explore-applications.ptx` from `explore-applications.qmd`)

### Part III: Regression modeling
7. Linear regression with a single predictor (`chapters/ch07-linear-regression-single.ptx` from `model-slr.qmd`)
8. Linear regression with multiple predictors (`chapters/ch08-linear-regression-multiple.ptx` from `model-mlr.qmd`)
9. Logistic regression (`chapters/ch09-logistic-regression.ptx` from `model-logistic.qmd`)
10. Applications: Model (`chapters/ch10-model-applications.ptx` from `model-applications.qmd`)

### Part IV: Foundations of inference
11. Hypothesis testing with randomization (`chapters/ch11-hypothesis-testing-randomization.ptx` from `foundations-randomization.qmd`)
12. Confidence intervals with bootstrapping (`chapters/ch12-confidence-intervals-bootstrapping.ptx` from `foundations-bootstrapping.qmd`)
13. Inference with mathematical models (`chapters/ch13-inference-mathematical-models.ptx` from `foundations-mathematical.qmd`)
14. Decision Errors (`chapters/ch14-decision-errors.ptx` from `foundations-errors.qmd`)
15. Applications: Foundations (`chapters/ch15-foundations-applications.ptx` from `foundations-applications.qmd`)

### Part V: Statistical inference
16. Inference for a single proportion (`chapters/ch16-inference-single-proportion.ptx` from `inference-one-prop.qmd`)
17. Inference for comparing two proportions (`chapters/ch17-inference-two-proportions.ptx` from `inference-two-props.qmd`)
18. Inference for two-way tables (`chapters/ch18-inference-two-way-tables.ptx` from `inference-tables.qmd`)
19. Inference for a single mean (`chapters/ch19-inference-single-mean.ptx` from `inference-one-mean.qmd`)
20. Inference for comparing two independent means (`chapters/ch20-inference-two-independent-means.ptx` from `inference-two-means.qmd`)
21. Inference for comparing paired means (`chapters/ch21-inference-paired-means.ptx` from `inference-paired-means.qmd`)
22. Inference for comparing many means (`chapters/ch22-inference-many-means.ptx` from `inference-many-means.qmd`)
23. Applications: Infer (`chapters/ch23-inference-applications.ptx` from `inference-applications.qmd`)

### Part VI: Inferential modeling
24. Inference for linear regression with a single predictor (`chapters/ch24-inference-linear-regression-single.ptx` from `inf-model-slr.qmd`)
25. Inference for linear regression with multiple predictors (`chapters/ch25-inference-linear-regression-multiple.ptx` from `inf-model-mlr.qmd`)
26. Inference for logistic regression (`chapters/ch26-inference-logistic-regression.ptx` from `inf-model-logistic.qmd`)
27. Applications: Model and infer (`chapters/ch27-model-infer-applications.ptx` from `inf-model-applications.qmd`)

### Part VII: Appendices
A. Exercise solutions (`appendix/exercise-solutions.ptx` from `exercise-solutions.qmd`)
