### Conversion Approach

#### Quarto to PreTeXt Mappings
- Bold text (`**text**`) → `<alert>text</alert>`
- Italic text (`*text*`) → `<em>text</em>`
- Inline code (`` `code` ``) → `<c>code</c>`
- Math (`$math$`) → `<m>math</m>`, (`$$...$$`) → `<me>...</me>` or `<md>...</md>`
- Figure references (`@fig-ref`) → `<xref ref="fig-ref" />`
- Section references (`@sec-ref`) → `<xref ref="sec-ref" />`
- Table references (`@tbl-ref`) → `<xref ref="tbl-ref" />`
- Chapter references (`[Chapter -@sec-ref]`) → `<xref ref="sec-ref" />`
- Bibliography (`[@Ref:Year]`) → `<xref ref="Ref:Year" />`
- URLs (`[text](url)`) → `<url href="url">text</url>`
- Chapter intro (`::: {.chapterintro}`) → `<introduction>`
- Guided practice (`::: {.guidedpractice}`) → `<exercise>` with `<statement>` and `<solution>`
- Worked examples (`::: {.workedexample}`) → `<example>` with `<statement>` and `<solution>`
- Data boxes (`::: {.data}`) → `<note><title>Data</title>`
- Important boxes (`::: {.important}`) → `<assemblage>`
- R code blocks → `<listing>` with `<program language="r">` for toggle capability

#### Special Features
- **R Code Toggle**: All R code blocks are wrapped in `<listing>` elements with captions linking to the generated figure. This makes them collapsible/toggleable by default (hidden like exercises), as requested in the issue.
- **Checkpoint Solutions**: All guided practice exercises include solutions under `<solution>` tags, similar to exercises.
- **Cross-references**: All references to figures, sections, tables, and other chapters are properly linked using `<xref>` tags.
